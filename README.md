# SafeRoute - AI-Powered Smart Safety Navigation

SafeRoute is a premium navigation platform designed to optimize for **safety** rather than just distance or time. It combines community intelligence, real-time reporting, and an emergency SOS system into a sleek, futuristic cyberpunk interface.

## Core Features

- **Interactive Map**: Powered by Leaflet.js with CartoDB Dark Matter tiles, featuring custom glowing markers and animated routes.
- **AI Safety Routing**: Graph-based routing that considers danger penalties (crime reports, poor lighting) and safety bonuses (police presence, CCTV).
- **Community Reporting**: Real-time reporting of safety conditions directly on the map.
- **Emergency SOS**: A high-priority alert system that captures GPS coordinates and notifies contacts.
- **Cyberpunk UI**: A modern, mobile-responsive interface utilizing glassmorphism and motion animations.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4, Motion, Lucide Icons.
- **Backend**: Express (Node.js) with Vite Middleware.
- **Database**: Firebase Firestore for real-time data sync.
- **Auth**: Firebase Google Authentication.
- **Map**: Leaflet.js with custom dark themes.

## How to use

1. **Sign In**: Use your Google account to access all features.
2. **Search**: Enter a destination in the sidebar to find the safest route.
3. **Report**: Click anywhere on the map to log a safety report (Danger or Safe Spot).
4. **Emergency**: Use the floating red SOS button in case of immediate danger.
